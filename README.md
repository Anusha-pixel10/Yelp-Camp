YelpCamp is a full-stack web application that allows users to explore, create, and review campgrounds online. Users can:

- View a list of all campgrounds with images and descriptions  
- Add new campgrounds with custom details  
- Edit or delete campgrounds theyâ€™ve created  
- Post reviews and ratings for each campground  

Tech Stack & implementation:

- Backend: Node.js, Express.js, Mongoose  
- Database: MongoDB (Atlas for deployment)  
- Frontend: EJS templates, HTML, CSS  
- Features Implemented:  
  - CRUD operations for campgrounds  
  - User reviews and ratings  
  - Server-side form validation  
  - Database seeding and deployment-ready integration  
  - Fully deployed on Render
